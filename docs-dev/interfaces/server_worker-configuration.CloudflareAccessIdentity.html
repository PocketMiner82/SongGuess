    Interface CloudflareAccessIdentity

    Represents the identity of a user authenticated via Cloudflare Access. This matches the result of calling /cdn-cgi/access/get-identity.

    The exact structure of the returned object depends on the identity provider configuration for the Access application. The fields below represent commonly available properties, but additional provider-specific fields may be present.

    interface CloudflareAccessIdentity {
        account_id?: string;
        amr?: string[];
        devicePosture?: Record<string, unknown>;

        iat?: number;
